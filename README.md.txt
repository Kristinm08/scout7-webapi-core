An example consist of ASP.NET Core 2 WebAPI and the ASP.NET Core MVC projects. 
The Scout7Api return Players data in json and Scout7Client MVC application consume 
and show the data in the table with sorting, filtering and paging functionality.

Setup:
1.Extract the Zip,
2.Build the solution using Visual Studio 2017, 
3.Run the projects. The Scout7Api project will start up on http://localhost:47285 and 
   Scout7Client project will start up on http://localhost:31050.
